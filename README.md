## 🚀 Getting Started

Clone o projeto

```
git clone https://github.com/RafaelGiro/5pots
```
Entre na branch dev

```
git checkout dev
```

Por enquanto só estamos desenvolvendo o Front então acesse a pasta `client`

```
cd client
```

Instale as dependências

```
yarn
```

Rode o projeto com

```
yarn start
```
